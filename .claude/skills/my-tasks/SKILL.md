# /skill-my-tasks — Мои задачи

Показывает текущие BPM-задачи пользователя из ELMA365.

## Алгоритм

1. Вызови `get_tasks` для получения списка задач
2. Отформатируй результат в удобную таблицу

## Формат ответа

```
## Мои задачи

| # | Задача | Статус | Срок |
|---|--------|--------|------|
| 1 | Согласование договора | В работе | 2025-03-15 |
| 2 | Проверка заявки | Новая | 2025-03-20 |

Всего задач: 2
```

## Примеры

```
/skill-my-tasks
/skill-my-tasks 0 10
```
